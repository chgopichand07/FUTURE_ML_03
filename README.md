# 📌 Project Overview

Recruiters receive hundreds of resumes for a single job opening, making manual screening tedious and time-consuming.

This project automates the recruitment process by analyzing resumes using NLP techniques. It extracts important skills, compares them with the required job description, calculates a matching score, ranks candidates according to their suitability, and identifies missing skills required for the role.

The system helps recruiters shortlist the most suitable candidates efficiently while reducing manual effort.
